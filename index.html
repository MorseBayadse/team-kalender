In externem Kalender abonnieren
Kopiere die URL unten und füge sie in deinem Outlook, Apple Kalender oder Google Kalender als Kalenderabonnement ein. Der externe Kalender zeigt dann alle Termine, an denen du beteiligt bist (read-only), und aktualisiert sich automatisch.
Behandle diesen Link wie ein Passwort – wer ihn hat, sieht deine Termine.
Anleitung: Outlook / Apple / Google
Outlook: Datei → Konto­einstellungen → Internetkalender → Neu → URL einfügen.
Apple Kalender (Mac): Ablage → Neues Kalender­abonnement → URL einfügen.
Apple Kalender (iOS): Einstellungen → Kalender → Accounts → Account hinzufügen → Anderer → Kalender­abo hinzufügen.
Google Kalender: Andere Kalender → + → Per URL → URL einfügen.
